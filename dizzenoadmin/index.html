Roles del sistema
Administrador — Acceso total. Gestiona usuarios, configuración, precios e inventario.

Editor — Crea y edita productos. Sin acceso a usuarios ni configuración.

Lector — Solo visualización del inventario.
Cómo agregar un usuario
Paso 1 — Firebase Console → Authentication → Add user → correo y contraseña.

Paso 2 — Aquí → + Nuevo usuario → nombre, correo y rol.

El usuario puede ingresar de inmediato con sus credenciales.
